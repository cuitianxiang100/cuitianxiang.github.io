#### 2，字典
数据类型划分：可变数据类型，不可变数据类型
不可变数据类型：元组，bool int str       可哈希
可变数据类型：list，dict set             不可哈希
dict key 必须是不可变数据类型，可哈希，
    value：任意数据类型。
dict 优点：二分查找去查询
         存储大量的关系型数据
      特点：无序的
